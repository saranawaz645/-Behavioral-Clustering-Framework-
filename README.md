📊 Behavioral Clustering Framework
Unsupervised Learning for Customer Persona Identification
This repository focuses on identifying distinct consumer segments by analyzing multi-dimensional behavioral data. The core objective is to apply mathematical clustering techniques to derive actionable insights from complex user profiles.

🔍 Analytical Approach
Data Normalization: Implemented Standard Scaling to ensure feature parity across Age, Annual Income, and Spending Score metrics.

Algorithmic Core: Utilized the K-Means Clustering algorithm to partition the dataset into statistically significant user groups.

Optimization Heuristics: Executed the Elbow Method to minimize the Within-Cluster Sum of Squares (WCSS), mathematically identifying the optimal value for 'K'.

Profiling: Derived 5 strategic segments representing unique economic behaviors (e.g., High-Value Consumers, Conservative Spenders).

🛠️ Technical Implementation
Language: Python 3.x

Core Libraries: scikit-learn, pandas, numpy

Mathematical Foundations: Euclidean Distance, Variance Minimization, and Heuristic Analysis.
